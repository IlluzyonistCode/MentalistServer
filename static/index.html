Installation Guide

Follow these simple steps to get Mentalist up and running

📁

Step 1: Create Installation Folder

Create a new folder anywhere on your PC where Mentalist will be installed. For example: C:\Mentalist or D:\Downloads\Mentalist

Note: Choose a location that's easy to remember and has at least 500MB of free space.
⬇️

Step 2: Download EXE File

Download your preferred version (CLI or GUI) from the Download section above and save it to the folder you created.

Go to Download Section
📝

Step 3: Create Configuration File

In the same folder where you saved the EXE, create a new text file named config.txt with the following template:

TRACKER_API_KEYS=YOUR_WOLVESVILLE_API_KEY
STALKER_API_KEYS=YOUR_WOLVESVILLE_API_KEY
CHROME_EXECUTABLE=C:\Program Files\Google\Chrome\Application\chrome.exe
CHROME_VIEWPORT=958,958
BLUESTACKS5_EXECUTABLE=C:\Program Files\BlueStacks_nxt\HD-Player.exe
BLUESTACKS5_NAME=Player
SERVER_SYNC_ENABLED=true
MENTALIST_SERVER_URL=https://mentalist.ydns.eu
MENTALIST_SERVER_API_KEY=YOUR_MENTALIST_API_KEY
⚙️

Step 4: Configure Parameters

Replace the placeholder values in config.txt with your actual settings:

TRACKER_API_KEYS & STALKER_API_KEYS

Required for Tracker & Stalker

To get API keys:

  1. Open Wolvesville game
  2. Go to Settings → Wolvesville Public API
  3. Purchase API key for 100 gems
  4. Copy the key and paste it in config.txt
Tip: You can use multiple keys from different accounts separated by commas for better efficiency.

CHROME_EXECUTABLE

Optional

Path to your Chrome browser executable. Leave default if Chrome is installed in the standard location.

CHROME_VIEWPORT

Optional

Browser window size. You can keep default 958,958 or get your viewport from whatismyviewport.com and enter as width,height

BLUESTACKS5_EXECUTABLE & BLUESTACKS5_NAME

Required for Spinner

Path to BlueStacks 5 executable and window name. Only needed if you want to use the Spinner module.

SERVER_SYNC_ENABLED

Optional

Enable cloud synchronization. Set to true to sync data across devices or false to keep data local.

MENTALIST_SERVER_URL & MENTALIST_SERVER_API_KEY

Required if sync enabled

Server URL and your personal API key for cloud sync. Contact admin via Instagram or Discord to get your key.

🚀

Step 5: Launch Mentalist

Double-click the EXE file to launch Mentalist. Select the module you want to use and enjoy!

✅

Installation Complete!

You're all set! If you encounter any issues, check the contact section for support.
